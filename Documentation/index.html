Get started

Installation:

Infinity is HTML template so there's no really "installation" required just unzip the main file you've downloaded from wrapbootstrap and after unziping you have your all template ready to be used.

Setting Up The Grid:

Before starting to work deeper with this template, you'll need to know how to use template's grid, it's simple and easy to use. Infinity is being build using one of the most popular Framwork, Twitter bootstrap, which means your site container is 12-column grids wide.To make it work without problem you be using columns for everything .

Working With Navigation:

I have used Multi-Level Push Menu with jQuery improved menu but the menu itself is easy to edit. Its structure base on a nested unorederd lists structure. This menu will open by pushing the website content to the right. It has multi-level functionality, allowing endless nesting of navigation elements.
