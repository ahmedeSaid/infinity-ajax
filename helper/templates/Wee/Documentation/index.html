A) HTML Structure - top

Wee is a responsive HTML5 perfect for businesses, freelances, agencies and corporates. Wee is built with the awesome framework Skeleton. It has big bold typeface and vibrant, clean design. Every page has following sections:

  • Header - Logo and top navigation
  • Content - This is where we see verything
  • Footer - Contains contact details, social links and some other information.

Home page has an additional section for static banner and image slider.

C) Template Customization - top

How to Change Logo

Find header, inside header tag you will find an h1 tag wrapping our logo. Change the text or replace with your logo image.

How to Change Menu Items and Structure

Inside the header tag, right below the logo, there is a tag called Nav, it contains the top menu. The menu items are wrapped with a li, change the text and corresponding html page to reflect the change. For dorpdown menu, you will see another ul inside an li tag. Something like below: 

<li><a href="javascript:">Projects</a>
	<ul>
		<li><a href="project-threecol.html">Three Cols </a></li>
		<li><a href="project-fourcol.html">Four Cols</a></li>
		<li><a href="project-details.html">Project Details</a></li>
	</ul>
</li>
Change the required menu items with their respective html pages.

Every HTML page is well commented and so as the CSS file. If you would like to edit a specific section of the site, simply find the appropriate label/class in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
